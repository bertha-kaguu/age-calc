🎂 Elite Age Calculator

A modern Age Calculator Web App built with HTML, CSS, and JavaScript that calculates a person's age and displays additional fun information like zodiac signs, birthday countdowns, and live age in seconds.

This project demonstrates DOM manipulation, LocalStorage usage, date calculations, UI design, and animations.

🚀 Features

👥 Multiple People Support
Add and track the ages of multiple people.

🌙 Dark / Light Mode
Toggle between themes with your preference saved in the browser.

⏳ Next Birthday Countdown
Displays how many days remain until the next birthday.

♈ Zodiac Sign Detection
Automatically calculates the zodiac sign from the birthdate.

🎉 Birthday Confetti Animation
A fun confetti animation triggers when today is someone's birthday.

⏱ Live Age in Seconds
Age updates every second showing how many seconds a person has lived.

📊 Birthday Progress Bar
Shows how far you are through the year until the next birthday.

💾 LocalStorage Persistence
Saved people remain even after refreshing the page.

⚙️ How It Works

Enter a name.

Select a birthdate.

Click Add.

The app calculates:

Years

Months

Days

Zodiac sign

Birthday countdown

Age in seconds

The information is saved locally in the browser using LocalStorage.
